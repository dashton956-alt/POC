üåê Overview

The Intent Based Orchestrator is a comprehensive network infrastructure management platform that seamlessly integrates with NetBox DCIM to provide automated device provisioning, workflow orchestration, and infrastructure management capabilities.

Key Features

  • NetBox Integration: Direct integration with NetBox for device and manufacturer management
  • Device Library Import: Automated import from community device-type library
  • Workflow Orchestration: Automated network configuration workflows
  • Real-time Monitoring: Live dashboard with system status and statistics

üì± Device Import Process

Import device types from the community device-type library with full component support.

Import Workflow Steps

  1. Navigate to Workflows: Click "Import Device Types" on the dashboard
  2. Select Manufacturers: Choose which manufacturers to import
  3. Monitor Progress: Watch real-time import progress
  4. Review Results: Check import summary and any errors
  5. Validate in NetBox: Verify devices appear in NetBox

Import Features

  • Intelligent Filtering: Only imports new devices, skips existing ones
  • Component Support: Includes interfaces, power ports, console ports, and module bays
  • Validation: Validates device definitions before import
  • Error Handling: Detailed error reporting for failed imports

üí° Best Practice

Start with a small set of manufacturers to test the import process, then expand to include more as needed.

Import Failures

  • Duplicate Devices: The system automatically skips existing devices
  • Invalid YAML: Check device-type library files for syntax errors
  • Permission Errors: Verify NetBox API token has write permissions
